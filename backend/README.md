# MIDINecromancer Backend

FastAPI backend for the MIDINecromancer generative MIDI composition workbench.

## Features

- RESTful API for project, track, clip, and note management
- Music theory-driven generation (polyrhythms, chord progressions, melodies)
- Theory overlay with suggestion engine
- MIDI export functionality
- PostgreSQL database with Alembic migrations

## Tech Stack

- Python 3.12+
- FastAPI
- SQLAlchemy 2.0 (async)
- Alembic for migrations
- uv for package management
